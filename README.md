# Country Service

Este microservicio proporciona una API REST para consultar información de países (nombre, capital, región y población).

## 🚀 Características

- Listado de países
- Ordenamiento alfabético

## 📦 Tecnologías

- Go 1.24.3
- net/http
- encoding/json
- Echo

## ⚙️ Instalación

```bash
git clone https://github.com/glasside/maatia.git
cd maatia
go mod tidy
go run server.go
